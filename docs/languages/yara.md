# Yara

`Yara` language support in Gram is provided by the [Yara language extension](https://github.com/egibs/yara.zed). Please report issues to [https://github.com/egibs/yara.gram/issues](https://github.com/egibs/yara.gram/issues).

- Tree-sitter: [egibs/tree-sitter-yara](https://github.com/egibs/tree-sitter-yara)
- Language Server: [avast/yls](https://github.com/avast/yls)
